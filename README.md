# Project Data and Code

This repository contains the data and scripts associated with the submitted work.

## Folder Contents:

-   **Figure_6**: Contains data files (e.g., `count0.csv`) and scripts (`acf.py`) used to generate Figure 6.
-   **Figure_7&Table_1**: Contains data organized into subdirectories (e.g., `V1.3_F1_T27/`) with binary files (`.bin`) and a drawing script (`draw.py`) related to Figure 7 and Table 1.
-   **Table_2**: Contains binary data files (e.g., `1.3V_1G.bin`), a Jupyter Notebook for summarization (`H_summary.ipynb`), and a CSV file (`parameter_table.csv`) pertinent to Table 2.
